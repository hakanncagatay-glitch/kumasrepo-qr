# 📱 KumasRepo QR

QR Kod okuma ve oluşturma uygulaması. React + Vite ile geliştirilmiştir.

## ✨ Özellikler

- 📖 **QR Kod Okuma**: Kamera ile QR kodları tarayın
- ✨ **QR Kod Oluşturma**: Metin veya URL'den QR kod oluşturun
- 💾 **İndirme**: Oluşturulan QR kodlarını PNG olarak indirin
- 📋 **Kopyala**: Metinleri kolayca kopyalayın
- 📱 **Responsive**: Mobil ve masaüstü cihazlarda çalışır

## 🚀 Kurulum

### Gereksinimler
- Node.js 16+
- npm veya yarn

### Adımlar

```bash
# Projeyi klonla
git clone https://github.com/hakanncagatay-glitch/kumasrepo-qr.git
cd kumasrepo-qr

# Bağımlılıkları yükle
npm install

# Geliştirme sunucusunu başlat
npm run dev

# Üretim için derle
npm run build
